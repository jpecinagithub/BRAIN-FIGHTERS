# Documentacion de Brain Fighters

Esta carpeta concentra la documentacion funcional y tecnica del proyecto.

## Indice
- overview.md: Vision general, roles y flujo principal.
- setup.md: Requisitos y pasos para correr el proyecto en local.
- backend.md: API, autenticacion, endpoints y configuracion del servidor.
- frontend.md: Rutas, vistas y consumo de API.
- database.md: Modelo de datos y relaciones.
- games.md: Catalogo de minijuegos y codigos.

## Como leerla
Empieza por `overview.md`, luego continua con `setup.md`. Si vas a desarrollar, revisa `backend.md`, `frontend.md` y `database.md`.
